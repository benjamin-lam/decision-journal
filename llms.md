# LLM Guidance

This repository contains a minimal experimental tool.

Principles:

KISS  
YAGNI  
Vanilla JavaScript  
Mobile first  
Local-first data storage

Purpose:

A private project journal for project managers.

The system records:

- decisions
- meeting notes
- risks
- context

Data is stored locally using IndexedDB.

No backend.

The goal is clarity and traceability of project history.
